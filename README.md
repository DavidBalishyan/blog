# David's Blog

A minimal Jekyll blog with a Gruvbox Dark theme.

## Quick Start

```bash
make install   # Install dependencies
make dev       # Start dev server at http://localhost:4000/blog/
```

## Commands

| Command         | Description                         |
|-----------------|-------------------------------------|
| `make build`    | Generate the static site in `_site` |
| `make serve`    | Start dev server with livereload    |
| `make dev`      | Alias for `serve`                   |
| `make clean`    | Remove build artifacts and cache    |
| `make install`  | Install Ruby dependencies           |
